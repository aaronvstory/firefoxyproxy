Setting Up FoxyProxy with 922Proxy

1

Install FoxyProxy Standard

First, you need to install the FoxyProxy Standard add-on for Firefox if you haven't already:

Get FoxyProxy Standard
2

Generate Proxy Configuration

Using the 922Proxy extension popup, click the "Generate New Proxies" button to create a set of unique proxy configurations.

3

Download the Configuration File

After generating the proxies, click the "Download Config" button to save the FoxyProxy-compatible JSON file to your computer.

4

Import the Configuration into FoxyProxy

Now open FoxyProxy and import the configuration:

  1. Click the FoxyProxy icon in your Firefox toolbar
  2. Select "Options" to open the configuration page
  3. Click on "Import Settings"
  4. Browse for and select the JSON file you downloaded
  5. Confirm the import when prompted
5

Set FoxyProxy Mode

Once imported, set FoxyProxy to use one of your imported proxies:

  1. Click the FoxyProxy icon in your Firefox toolbar
  2. You'll see your list of imported proxies
  3. Click on the proxy you want to use

Important Tips

  • Each proxy gives you a unique USA residential IP address
  • You can switch between proxies anytime by clicking a different one
  • To see your current IP, visit ipinfo.io/what-is-my-ip

Using Containers with FoxyProxy

Firefox Multi-Account Containers let you separate your online identities, and when combined with FoxyProxy, you can have different IPs for each container.

1

Create a New Container

From the 922Proxy extension popup, click the "New Container" button to create a Firefox container.

2

Switch Proxy Before Using Container

Before browsing in your container, click the FoxyProxy icon and select a proxy from your list.

Pro Tip

You can associate specific proxies with specific containers using FoxyProxy's "Patterns" feature (for advanced users).
